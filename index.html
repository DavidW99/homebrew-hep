Homebrew-hep

High energy physics packages for Mac

View project onGitHub

Introduction

Many high energy physics programs require special installation instructions for Mac OS X, and their issue trackers are often filled with unresolved Mac issues. Problems are also encountered when trying to link all these programs together.

Homebrew-hep is a Mac package manager for HEP programs, based on Homebrew. In Homebrew language, it is a tap. From the end-user's perspective it is nice because it:

  • keeps things organised in /usr/local
  • automatically handles package dependencies
  • is easy to update programs

Instructions

  1. Install Homebrew: ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"
  2. Tap homebrew-hep: brew tap davidchall/hep
  3. Install HEP packages: brew install <package> (e.g. brew install pythia8)

A list of available HEP packages can be found here.

Documentation

  • Homebrew: brew help, man brew, or the Homebrew wiki
  • HEP package: brew info <package>

Help

If you are experiencing problems with a specific package, please report it on the issue tracker. Please note that this is intended for issues arising from Mac installation, not problems with the package itself.

Can I contribute?

Of course! Some possible ways could be:

  • add a new package
  • add a new option to an existing package
  • update a package to the latest version

Please see the Homebrew-hep wiki for full instructions on how you can help.
